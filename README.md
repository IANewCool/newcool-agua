# Agua Chile - NewCooltura Informada

Modulo de informacion sobre recursos hidricos y derechos de agua en Chile. Parte del ecosistema NewCool.

## Funcionalidades

- **Buscador DGA**: 16 oficinas regionales de la Direccion General de Aguas
- **Tipos de Derechos**: Consuntivo, no consuntivo, permanente, eventual
- **Cuencas Principales**: Loa, Copiapo, Elqui, Aconcagua, Maipo, Maule, Biobio, Baker
- **Proceso de Obtencion**: 6 pasos para obtener derechos de agua
- **Calculadora de Consumo**: Estima costo de agua potable
- **Glosario Hidrico**: Terminos tecnicos explicados

## Tech Stack

- Next.js 16
- React 19
- Tailwind CSS 4
- Framer Motion

## Desarrollo

```bash
npm install
npm run dev
```

## Datos

Informacion basada en datos oficiales de la DGA (dga.mop.gob.cl).
